# 非周期型用户规则（绝经 / 围绝经 / 哺乳 / 备孕）

`profile.cycle.status ≠ regular` 时，**完全不读** `cycle-rules.md`，改读本文件。LLM 不要再说"你现在是黄体期"等 phase-aware 话术。

## 路由分流

| `cycle.status` | 走哪个章节 |
|---|---|
| `regular` / `irregular` | `cycle-rules.md`（不规律时 phase 推算可能不准，提示用户即可） |
| `peri_menopause` | 本文件 §1 围绝经期 |
| `menopause` | 本文件 §2 绝经后 |
| `breastfeeding` / `pregnant` | 本文件 §3 仅安全建议 + 强烈推医 |
| `medical` | 本文件 §4 个体化处理 |

## §1 围绝经期（peri_menopause）

**生理背景**：通常 40-55 岁，雌激素剧烈波动（不是单纯下降），月经间隔不稳，潮热 / 失眠 / 情绪波动 / 关节不适常见。**这一阶段比绝经后更难受**，因为体内激素像过山车。

### 饮食重点（4-5 条）

- **高蛋白**：1.2-1.6 g/kg 体重（同 follicular），抗肌肉流失加速
- **钙 + 维生素 D**：每天 2 杯奶 / 一片黑芝麻 / 一份深绿叶菜 + 适当晒太阳；骨密度从这一阶段开始下降
- **大豆异黄酮**：豆腐、豆浆、毛豆每天一份，植物雌激素温和缓冲潮热
- **减咖啡 + 减酒精**：会加重潮热和睡眠问题；每日 ≤ 1 杯咖啡，14:00 后不喝
- **抗炎食物**：深海鱼（三文鱼 / 鳕鱼）每周 2 次、坚果（核桃 / 杏仁）一小把 / 天、莓果、姜黄

### 训练强度（4-5 条）

- **力量训练优先**：每周 3 次抗阻（用 `weekly-split.md` 模板 B 上下半身 split），强骨 + 维肌
- **中等有氧 150 min/周**：快走 / 椭圆 / 骑行 / 游泳，分散到 3-5 天
- **避免极限高强度**：HIIT 减半（皮质醇本就高，加剧潮热和失眠）
- **平衡训练每周 1 次**：单腿站立、太极、瑜伽流——预防跌倒
- **关节友好优先**：跳跃 / 波比跳 / 深蹲跳谨慎；用 walking lunge 替代 jump lunge

### 情绪 + 睡眠应对

- 情绪波动是激素事实，不是个人意志问题（同 luteal 期理念，但更剧烈）
- 睡眠优先级 ≥ 训练：保证 7-8 小时；潮热夜醒后不要硬扛刷手机
- 不在 19:00 后做高强度训练（升高体温加剧夜间潮热）
- 必要时建议看妇科 / 内分泌科评估激素替代治疗（HRT）—— 本 skill 不诊断，只建议就医

## §2 绝经后（menopause，连续 ≥ 12 月无月经）

**生理背景**：通常 50+ 岁。雌激素稳定在低位。**风险**：骨密度下降、内脏脂肪倾向增加、肌肉流失加速（sarcopenia）、心血管风险升、皮肤胶原蛋白流失。**机会**：身体节奏稳定，不再被 phase 起伏干扰，**长期执行力比年轻人更高**。

### 饮食重点（4-5 条）

- **更高蛋白**：1.4-1.8 g/kg（高于围绝经），抗 sarcopenia
- **钙摄入 1000-1200 mg/天**：等于 2-3 份奶制品 + 深绿叶菜；骨健康优先
- **维生素 D 600-800 IU/天**：晒太阳 + 适量补剂（建议医生指导）
- **减加工糖 + 减精制碳水**：内脏脂肪倾向已经升高，糖摄入增重更直接
- **充足水分 1.5-2 L/天**：代谢减缓，肾负担同步降但水分摄入仍重要

### 训练强度（4-5 条）

- **力量训练 = 重中之重**：每周 3 次（推 `weekly-split.md` 模板 B 或 C），渐进负重
- **复合动作首选**：深蹲 / 硬拉变式 / 划船 / 推举——多关节动作刺激最高效
- **中等有氧 150 min/周**：心血管保护
- **平衡 + 灵活性每周 2 次**：太极 / 普拉提 / 瑜伽——预防跌倒和驼背
- **不需要 phase 节奏**：训练强度按主观恢复感调整，但同肌群隔 48 小时

### 减脂目标的特别说明

- 减脂仍可行，但**速度比育龄女性慢约 30%**：默认 deficit 10%（不是 15%），按 `kcal-targets.md` 公式 × 0.90
- **不追求体重数字**：内脏脂肪 / 腰围 / 体能（爬楼不喘）/ 力量（深蹲重量）才是真指标
- BMI 25-27 区间对绝经后女性反而是健康保护区间——不要硬推到 BMI 22

### 心血管 + 骨健康监测建议

- **建议每年体检**：血脂、血糖、骨密度（DEXA）、女性激素相关指标
- 出现任何胸闷 / 头晕 / 异常出血 / 骨折 → 立即就医

## §3 哺乳 / 备孕 / 孕期（v0.0 不深入覆盖）

任何 `cycle.status` 命中本节的用户：

1. **强烈建议先咨询主治医生 / 产科医生**再开始任何减脂或训练计划
2. 本 skill **不给具体减脂方案**——哺乳期热量需求 +400-500 kcal、孕期更复杂、备孕期不适合赤字
3. 可给的支持：
   - 一般生活方式建议（睡眠 / 心理 / 散步类温和活动）
   - 倾听 + 共情（产后焦虑 / 体型变化压力 / 备孕焦虑）
4. **绝对禁项**：
   - 任何 < 维持热量的方案
   - 高强度 HIIT
   - 力量负重 ≥ 体重 50% 的复合动作（除非用户原本就是力量训练运动员，且已咨询医生）
   - 任何减肥药 / 代餐建议（无论何时）

## §4 medical（子宫切除 / 内分泌治疗 / 化疗等）

完全个体化。本 skill 仅提供：
- 一般生活方式建议
- 共情 + 信息整理（帮用户梳理跟医生该问的问题）
- **绝不给具体方案**——必须以主治医生的医嘱为唯一依据

## 与其他文件的协作

- `cycle-rules.md` — 仅 `regular` / `irregular` 用户读
- `weekly-split.md` — `peri_menopause` 用模板 B；`menopause` 用模板 B 或 C；§按 phase 调整 整段不适用，跳过
- `kcal-targets.md` — `menopause` 用户 deficit 改 10%（× 0.90）；其它非周期型用户按医生建议
- `lifestyle-tips.md` — 通用，但 menopause 用户对加工糖更敏感，提醒比常规多
- `exercises.md` — 禁忌映射额外加：50+ 用户优先低冲击；骨密度低 → 跳跃类动作慎用
- `reports.md` — 三份报告均需把开头描述从"目前在 [phase 中文]"改为"目前 `[status 中文]`，咱们走非周期路径"；Report 1 §3 分期重点的"经期适配"改为"无周期切换，按主观恢复感调整"
