# 一周训练框架（split + 有氧 + 空腹有氧）

> **适用提示**：本文件主要服务 `cycle.status ∈ {regular, irregular}` 用户。绝经 / 围绝经 / 哺乳 / 备孕用户的 split 选择见 `non-cyclic-rules.md` §1 §2，**§按 phase 调整 整段对她们不适用**。

本文件给 LLM 在用户问"给我这周的训练计划"时，按 **N 力量日 + 2-3 有氧日 + 1-2 休息日 / 主动恢复** 的框架排课。

**使用前提**：先读 profile，按 `cycle-rules.md` 当前 phase 决定整体强度区间，按 `exercises.md §禁忌动作映射` 过滤动作，**再按下面 §按 training.history 选 split 决策表**确定 split 模板，最后落到具体动作。

## 按 training.history 选 split 决策表（必读）

**LLM 在排周表前必须先读 `profile.training.history` 决定走哪个 split 模板**。给"有运动基础"用户硬塞全身循环 = 严重失配。

| `training.history` | 用户描述 | 推荐 split 模板 | 训练频率 | 单次时长 |
|---|---|---|---|---|
| `none` | 无运动习惯 | **全身循环**（一次性练全身大肌群，每次 4-6 个复合动作 ×2-3 组） | 2-3 次/周 | 30-40 min |
| `occasional` | 偶尔练 | **全身循环 + 1 个主题日**（如周一全身、周四下肢专项 / 上肢专项） | 3-4 次/周 | 30-45 min |
| `regular_1-2x` | 每周 1-2 次规律训练 | **上下半身 split（2-day split）**：上半身日（背 + 肩 + 胸）+ 下半身日（臀 + 腿 + 核心） | 3-4 次/周 | 45-60 min |
| `regular_3+x` | 每周 3+ 次规律训练 | **3-day body-part split**：臀腿日 / 肩背日 / 胸核心日（本文件下方默认） | 4-5 次/周 | 45-60 min |
| 用户未填 / null | — | 默认 `regular_1-2x` 上下半身 split + 一句"如果你之前每周练 3+ 次，告诉我可以升级到 3 天 split" | 3-4 次/周 | 45-60 min |

**特殊覆盖**：
- 用户主动说"我练了 2 年""能做 10 个引体向上"等 → 直接按 `regular_3+x` 走
- 用户主动说"完全没练过""产后第一次"→ 即使 history 是其它值，按 `none` 走 + 加一句"先适应 2-3 周再升级"
- 当前 phase = `menstrual` D1-D3 → 任何 split 都暂停，仅推散步 / 阴瑜伽

**升级节奏**：
- `none` → `occasional`：连续 4-6 周稳定执行后
- `occasional` → `regular_1-2x`：再 4-6 周
- `regular_1-2x` → `regular_3+x`：再 6-8 周

不要让用户"一上来就 3-day split 力量怪练"——这是私教真实建议节奏。

---

## 模板 A：全身循环（针对 `none` / `occasional`）

每次训练涵盖大肌群，避免"今天只练胸明天只练腿"的复杂分配。结构：

- 热身 5 min（动态拉伸 / 开合跳轻版 / 弓步走）
- 主体 4-6 个复合动作 ×2-3 组：
  1. 下肢主导：自重深蹲 / 弓步走 / 桥式（选 1-2）
  2. 上肢推：跪姿俯卧撑 / 哑铃肩推（选 1）
  3. 上肢拉：哑铃俯身划船 / 弹力带反向飞鸟 / 超人式（选 1）
  4. 核心：死虫子 / 鸟狗式 / 平板（选 1，黄体期组数减半）
- 拉伸 5 min

每个动作 10-15 次（自重）/ 8-12 次（负重）。组间休息 60-90 秒。

## 模板 B：上下半身 split（针对 `regular_1-2x`）

### 上半身日（45-60 min）

涵盖背 + 肩 + 胸 + 二头三头核心：
- 拉类 2 个：哑铃俯身划船 + 弹力带反向飞鸟 / 超人式
- 推类 2 个：跪姿俯卧撑 + 哑铃肩推
- 二头三头 1 个：哑铃二头弯举 / 三头臂屈伸
- 核心 1-2 个：死虫子 / 平板

### 下半身日（45-60 min）

涵盖臀 + 腿 + 核心：
- 复合 2 个：深蹲（自重 / 哑铃 / 高脚杯）+ 弓步（自重 / 哑铃）
- 臀肌孤立 1-2 个：桥式 / 单腿臀桥 / 蚌式
- 后链 1 个：直腿哑铃硬拉
- 核心 1 个：鸟狗式 / 死虫子

## 模板 C：3-day body-part split：臀腿日 / 肩背日 / 胸核心日（针对 `regular_3+x`）

### 臀腿日（45-60 min）

主目标：臀大肌 + 腘绳肌 + 股四头 + 内收肌

可选动作池（按 `home_equipment` / `gym_access` 选，禁忌过滤后）：
- 复合：深蹲（自重 / 哑铃 / 高脚杯）/ 弓步（哑铃 / 自重）/ 直腿哑铃硬拉
- 臀肌孤立：桥式 / 单腿臀桥 / 蚌式 / 侧平板抬腿
- 进阶（卵泡期 / 排卵期）：保加利亚分腿蹲、相扑深蹲、壶铃摆动

典型组合：4 个动作 × 3-4 组 × 10-15 次

### 肩背日（45-60 min）

主目标：背阔肌 + 菱形肌 + 后三角 + 肩袖稳定（特别针对久坐圆肩）

可选动作池：
- 拉类（背为主）：哑铃俯身划船 / 单臂哑铃划船 / 弹力带反向飞鸟 / 超人式
- 推类（肩为主，无肩伤前提）：哑铃肩推 / 阿诺德推
- 稳定 / 体态修正：YTW 字母臂动作 / 弹力带肩外旋
- 强化背：墙俯卧撑变体

典型组合：4-5 个动作 × 3 组 × 10-12 次。**重点是动作质量 + 肩胛收缩感**，而不是负重。

### 胸核心日（30-45 min）

主目标：胸大肌 + 核心（腹横肌 / 腹直肌 / 腹斜肌 / 多裂肌）

可选动作池：
- 胸：跪姿俯卧撑 / 标准俯卧撑（无肩伤前提）/ 哑铃胸推 / 哑铃飞鸟
- 核心：平板支撑 / 死虫子 / 鸟狗式 / 卷腹（无腰伤前提）/ 侧平板

典型组合：胸 2-3 个 + 核心 3-4 个 × 3 组。**黄体期把核心组数减半**（详见下方 phase 调整）。

## 默认一周框架（适用于模板 C / `regular_3+x`）

```
周一  臀腿日（力量主）
周二  有氧 30-40 min（HIIT 或稳态选一）
周三  肩背日（力量主）
周四  有氧 30-40 min 或 主动恢复（瑜伽 / 散步）
周五  胸核心日
周六  长有氧 60-90 min（户外快走 / 骑行 / 游泳 / 爬楼）或 户外活动
周日  全休 / 拉伸 / 阴瑜伽
```

如用户每周可训练天数少于 6 次，按以下顺序砍：
1. 砍周二的有氧（合并到周一末尾 10 min）
2. 砍周六长有氧（减为 30 min）
3. 实在忙：3 力量日 + 1 有氧 + 3 休息也可以接受

## 按 phase 调整

### menstrual（月经期 D1 ~ avg_period_days）

- D1-D3：**所有 split 日全部改成散步 / 阴瑜伽**，不要硬上力量
- D4-5（出血明显减少）：可恢复 1 次轻强度臀腿日（重量降 30%、组数减半）
- 全程不推 HIIT、不推核心训练
- 不推空腹有氧

### follicular（卵泡期 D5/6 ~ D13）

- 这是黄金期，按默认框架满血执行
- 力量动作可上"接近个人上限"重量
- HIIT 一周 2 次（如周二 + 周四）
- **空腹有氧：可推**（详见下方）

### ovulatory（排卵期 D14 ~ D16）

- 力量爆发期，是 PR 日 —— 臀腿/肩背/胸核心都可挑战重量
- 跳跃训练效果最佳（深蹲跳、跳箱），但**有膝盖禁忌的不推**
- HIIT 可加到 3 次/周
- 空腹有氧：可推
- ⚠️ 韧带相对松弛，热身延长到 10 min

### luteal（黄体期 D17 ~ avg_cycle_days）

- 力量训练 **重量降 10-20% / 组数 -1**
- HIIT **时长或频率减半**（如平时 2 次→ 1 次）
- 胸核心日的核心部分**组数减半**（黄体期核心训练易诱发腹胀加重）
- 长有氧改成中等强度（避免极限）
- **空腹有氧：不推**（黄体期皮质醇本就高，空腹高强度会加重水肿和情绪波动）

## 空腹有氧（morning fasted cardio）使用规范

### 适合人群（必须全部满足）

- ✅ 当前 phase ∈ {follicular, ovulatory}（卵泡 / 排卵期）
- ✅ 没有低血糖史 / 不在用降糖药 / 不在备孕 / 不在哺乳
- ✅ 起床到运动间隔 ≤ 30 分钟（不要拖到上午 10 点还没吃饭再去跑）
- ✅ 用户主动愿意（不强推）

### 不适合（任一命中即否决）

- ❌ menstrual（月经期）
- ❌ luteal（黄体期）
- ❌ 任何"反复头晕 / 起床心慌 / 早饭不吃就手抖"的用户
- ❌ BMI < 19 或 `body_fat_pct < 18` 的用户（脂肪储备不足，不需要靠空腹再压低）
- ❌ 慢性病 / 长期服药用户（先咨询医生）

### 推荐时长 / 强度

- **20-30 分钟**（不超过 30 min；超过会启动肌肉分解）
- **强度低-中**：心率控制在最大心率的 60-70%（约 "能轻松对话" 到 "短句对话" 之间）
- 适合形式：快走、慢跑、爬楼、室内骑行、低速椭圆机
- **不推荐**：HIIT、空腹力量训练、长跑（≥45 min）

### 操作步骤建议

1. 起床后 5 分钟内喝 300-500 mL 温水
2. 运动前可少量喝黑咖啡（无糖无奶，约 100 mg 咖啡因）—— 提神且对空腹脂肪氧化略有帮助
3. 运动后 15-30 分钟内吃早餐：蛋白 25-35g + 复合碳水 + 一小份蔬果
4. 不要"运动完很饿就胡乱吃"—— 提前准备好早餐

### 长期使用建议

- 一周 2-3 次足够，**不要天天空腹**（皮质醇长期升高反而促脂肪堆积）
- 月经异常 / 周期变长 / 经期 PMS 加重 = 立即停止空腹有氧 + 评估是否过度训练

## 复述确认时的展示格式建议

LLM 输出本周训练计划时，每天用一行写明 + 时长 + 主动作 2-3 个（参考 `exercises.md` 的动作清单）。**仅文字，不附示范图 / URL**。例：

```
**周一 臀腿日（45 min，luteal 中等强度）**

- 自重深蹲 ×12 ×3 组
- 单腿臀桥 ×15/腿 ×3 组
- 哑铃弓步（5kg）×10/腿 ×3 组
- 死虫子 ×12/侧 ×2 组（黄体期减半）

**周二 主动恢复（30 min）**

- 户外快走 30 min（黄体期推荐稳态有氧，不推空腹）
```

避免输出大段表格 / 鸡汤式总结。每天 4-6 行，用户看着能直接开练。需要看动作示范时引导用户搜 B 站 / 抖音 / Keep。
