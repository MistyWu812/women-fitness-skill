# 生活化建议库

本文件是 LLM 在产生具体食物或训练推荐时的参考库，目的是把"减脂"翻译成用户日常真实可执行的选项——便利店、外卖、家里客厅——而不是默认"鸡胸肉 + 西兰花 + 健身房"。下列便利店组合、外卖点法、训练菜单都是参考组合，LLM 应结合用户当前周期阶段（参考 `cycle-rules.md`）和 profile（饮食禁忌、做饭比例、训练经验、伤病史，参考 `state/profile-schema.md`）做最终筛选。营养数据为基于 2025 年中国大陆常见 SKU 的估算，引用时请加"约""估算""大致"等限定词，避免给用户造成精确卡路里的误导。

## B. 外卖点法（8 类品牌/菜系）

### B.1 黄焖鸡米饭
**减脂友好点法**：
- **要**：鸡腿肉去皮 + 笋片香菇 + 半份米饭（或不要米饭，另加一份青菜）
- **不要**：勾芡浓汁 / 加肠 / 油辣子 / 油豆腐 / 大份米饭
- **加**：一份炒时蔬（少油备注）/ 番茄蛋汤

参考 kcal：单人份 550-700 kcal（按上述点法），蛋白质 ~30 g
适配阶段：menstrual / luteal 优先（有热汤、温食、铁与蛋白），follicular / ovulatory 也可

### B.2 麻辣烫
**减脂友好点法**：
- **要**：清汤底（番茄/菌菇）/ 多绿叶菜（生菜、菠菜、油麦菜）/ 蛋白质（鸡胸/牛肉/虾滑/鱼豆腐）/ 魔芋丝代粉
- **不要**：麻酱小料 / 油炸面筋 / 火腿肠 / 蟹棒 / 酥肉 / 宽粉大份
- **加**：一份卤蛋或茶叶蛋

参考 kcal：450-650 kcal（清汤+少粉做法），蛋白质 ~30 g
适配阶段：四个阶段都适配；luteal/menstrual 选清汤+热汤暖腹

### B.3 沙县小吃
**减脂友好点法**：
- **要**：清汤面（去油花）/ 蒸饺（瘦肉款，半份）/ 拌青菜 / 鸡腿饭去皮 + 半份饭
- **不要**：花生酱拌面 / 卤大肠 / 炒粉 / 飘满油的炖罐
- **加**：一份炖蛋或一份青菜汤

参考 kcal：400-600 kcal，蛋白质 ~22 g
适配阶段：所有阶段；menstrual 推荐炖罐（乌鸡/排骨）补铁补汤

### B.4 西贝莜面村
**减脂友好点法**：
- **要**：莜面鱼鱼（番茄汁/羊肉清汤款）/ 烤鱼（柠檬款，少油）/ 凉拌沙葱 / 蒸南瓜
- **不要**：油泼莜面 / 大份炒莜面 / 黄馍馍带糖 / 油炸糕
- **加**：一份白灼时蔬

参考 kcal：500-700 kcal，蛋白质 ~28 g
适配阶段：所有阶段；莜面是优质粗粮，luteal 阶段适合用来稳定血糖

### B.5 肯德基
**减脂友好点法**：
- **要**：嫩牛五方（去酱一半）/ 烤鸡腿堡（不要蛋黄酱）/ 玉米杯（不加黄油）/ 鸡胸肉沙拉（油醋汁分开）
- **不要**：薯条 / 上校鸡块 / 香辣鸡腿堡 / 蛋挞 / 含糖可乐
- **加**：玉米一根 + 无糖茶/黑咖啡

参考 kcal：450-600 kcal，蛋白质 ~28 g
适配阶段：follicular / ovulatory 偶尔可，menstrual / luteal 不主推（油盐偏高易水肿）

### B.6 星巴克 / 瑞幸 / Manner
**减脂友好点法**：
- **要**：美式 / 拿铁（燕麦奶/低脂奶替换）/ 无糖冷萃 / 鸡胸三明治轻食 / 全麦贝果
- **不要**：星冰乐 / 焦糖玛奇朵 / 厚乳系列默认糖 / 司康配奶油 / 加奶盖
- **加**：一颗卤蛋或一份酸奶坚果杯

参考 kcal：饮品 ~80-180 kcal（按上法），轻食 300-450 kcal
适配阶段：所有阶段；luteal 阶段建议拿铁去冰、避免空腹冷饮

### B.7 必胜客
**减脂友好点法**：
- **要**：薄底披萨（鸡肉/海鲜款，1-2 块）/ 田园沙拉（油醋）/ 蘑菇浓汤换番茄汤 / 烤鸡翅小份
- **不要**：芝心 / 厚底 / 肉酱意面大份 / 含糖汽水 / 提拉米苏
- **加**：一份蔬菜沙拉

参考 kcal：500-700 kcal（控制 2 块薄底 + 沙拉），蛋白质 ~25 g
适配阶段：follicular / ovulatory 适合社交场景；luteal/menstrual 谨慎（盐分高易水肿）

### B.8 日料寿司
**减脂友好点法**：
- **要**：刺身拼盘 / 三文鱼/金枪鱼握寿司（4-6 贯）/ 茶碗蒸 / 海藻沙拉 / 味噌汤
- **不要**：天妇罗 / 炸猪排饭 / 蛋黄酱卷 / 鳗鱼饭大份酱汁 / 寿喜烧（糖盐高）
- **加**：一份毛豆 + 无糖绿茶

参考 kcal：450-650 kcal，蛋白质 ~30 g
适配阶段：所有阶段；menstrual 推荐三文鱼+味噌汤（Omega-3 + 暖汤）

## C. 家庭徒手训练菜单（3 时长 × 3 强度 = 9 套）

强度判定锚点：低 = 心率 ≤ 110 / 出汗少 / 轻松对话；中 = 心率 110-140 / 微汗 / 短句对话；高 = 心率 ≥ 140 / 大汗 / 不能完整说话。所有菜单皆徒手、无器械、不扰民为优先。

### C.1 10 分钟 - 低强度（适配 menstrual / luteal）
**热身**：颈部+肩部环绕 1 分钟 + 深呼吸 1 分钟（共 2 分钟）
**主体**：
1. 猫牛式 ×8 次（慢）
2. 婴儿式静态保持 30 秒 ×2
3. 束角式 + 前屈 1 分钟
4. 仰卧抱膝摇摆 1 分钟
5. 鸽子式左右各 1 分钟
**放松**：靠墙倒腿 1 分钟
**注意**：完全不冲击关节，经期第 1-2 天也可执行；任何动作疼痛立即停止。

### C.2 10 分钟 - 中强度（适配 follicular / ovulatory / luteal）
**热身**：原地踏步 1 分钟 + 动态侧弓步 1 分钟
**主体**：
1. 自重深蹲 ×15 ×2 组
2. 跪姿俯卧撑 ×10 ×2 组
3. 臀桥 ×15 ×2 组
4. 平板支撑 30 秒 ×2 组
5. 仰卧卷腹 ×15 ×2 组
**放松**：站立前屈 + 侧拉伸 1 分钟
**注意**：组间休息 30-45 秒；luteal 期酌情减一组。

### C.3 10 分钟 - 高强度（适配 follicular / ovulatory）
**热身**：开合跳 30 秒 + 高抬腿 30 秒（动态拉伸 1 分钟）
**主体**（Tabata 式：20 秒做 / 10 秒休 ×8 轮，配合下列动作循环）：
1. 波比跳
2. 深蹲跳
3. 登山者
4. 开合跳
**放松**：站立深呼吸 + 大腿后侧拉伸 1 分钟
**注意**：膝盖/腰部任何不适立刻降级到 C.2；menstrual / luteal 不建议。

### C.4 15 分钟 - 低强度（适配 menstrual / luteal）
**热身**：3 分钟温和瑜伽流（猫牛 + 下犬 + 婴儿）
**主体**：
1. 慢速深蹲（仅至椅子高度）×10 ×2 组
2. 靠墙静蹲 30 秒 ×3 组
3. 仰卧抬腿 ×10 ×2 组
4. 桥式静态保持 30 秒 ×3 组
5. 侧躺蚌式 ×12/侧 ×2 组
6. 鸽子式各 1 分钟
**放松**：仰卧扭转 + 大休息 2 分钟
**注意**：menstrual 第 1-3 天若觉疲惫可只做 C.1；动作以"舒展"非"消耗"为目标。

### C.5 15 分钟 - 中强度（适配 follicular / ovulatory / luteal）
**热身**：开合跳（小幅）1 分钟 + 动态弓步 1 分钟 + 肩部环绕 1 分钟
**主体**：
1. 自重深蹲 ×15 ×3 组
2. 弓步走 ×10/腿 ×3 组
3. 跪姿俯卧撑 ×10 ×3 组
4. 平板支撑 40 秒 ×3 组
5. 臀桥 ×15 ×3 组
6. 死虫子 ×10/侧 ×2 组
**放松**：股四头+腘绳肌+臀部静态拉伸 共 2 分钟
**注意**：组间 30-45 秒；动作出现代偿（膝内扣/腰塌陷）立即降负荷。

### C.6 15 分钟 - 高强度（适配 follicular / ovulatory）
**热身**：开合跳 1 分钟 + 高抬腿 1 分钟 + 动态拉伸 1 分钟
**主体**（HIIT 间歇：每动作 40 秒做 / 20 秒休，2 个循环）：
1. 波比跳
2. 深蹲跳
3. 俯卧撑（标准或跪姿）
4. 登山者
5. 平板交替提膝
6. 开合跳
**放松**：缓步走 + 全身拉伸 2 分钟
**注意**：心率长期 ≥ 160 即降级；不在 menstrual / luteal 后段推；3 个月内伤病史用户跳过。

### C.7 20 分钟 - 低强度（适配 menstrual / luteal）
**热身**：4 分钟流瑜伽（拜日式 A 慢速 ×3）
**主体**：
1. 慢速深蹲 ×12 ×3 组
2. 靠墙静蹲 45 秒 ×3 组
3. 桥式 ×15 ×3 组
4. 侧躺抬腿 ×12/侧 ×2 组
5. 死虫子 ×10/侧 ×2 组
6. 鸽子式 + 婴儿式 各 1 分钟
**放松**：仰卧扭转 + 靠墙倒腿 共 3 分钟
**注意**：菜单设计为 menstrual / luteal 期"安全运动剂量"，疲惫时可以中途切到 C.1 收尾。

### C.8 20 分钟 - 中强度（适配 follicular / ovulatory / luteal）
**热身**：3 分钟动态热身（开合跳 + 弓步 + 肩绕）
**主体**：
1. 自重深蹲 ×15 ×4 组
2. 弓步 ×12/腿 ×3 组
3. 跪姿（或标准）俯卧撑 ×12 ×3 组
4. 臀桥 ×15 ×4 组
5. 平板支撑 45 秒 ×3 组
6. 卷腹 ×15 ×3 组
7. 死虫子 ×10/侧 ×3 组
**放松**：腿部+核心+肩部静态拉伸 3 分钟
**注意**：组间 30-45 秒；可作为 follicular / ovulatory / 早期 luteal 阶段日常主力训练。

### C.9 20 分钟 - 高强度（适配 follicular / ovulatory）
**热身**：5 分钟（开合跳 + 高抬腿 + 动态弓步）
**主体**（HIIT 间歇：每动作 45 秒做 / 15 秒休，3 个循环）：
1. 波比跳
2. 深蹲跳
3. 登山者
4. 俯卧撑+提膝
5. 平板侧转
6. 开合跳
**放松**：缓步走 3 分钟 + 全身静态拉伸 2 分钟
**注意**：仅 follicular / ovulatory 推荐；任何头晕、关节痛立即停；3 个月内伤病/怀孕史/心血管问题禁做，建议医师面诊后再决定。

## 个性化优先级（强制读 profile 再推荐）

LLM 在生成饮食建议（早午晚加餐）前，**必须按以下顺序检查 profile**：

1. **首选 `profile.diet_pattern.pantry_ingredients`** —— 用户家里已有的食材
   - 例：用户答了"燕麦 / 鸡蛋 / 紫薯 / 鸡胸 / 西兰花" → 早餐推 "燕麦杯 + 鸡蛋 ×2 + 蓝莓"，午餐推 "紫薯 + 鸡胸 + 西兰花"
   - **不要**推用户没填的食材作为首选（如鳕鱼、藜麦、奇亚籽，除非她明确写了）

2. **外卖场景首选 `profile.diet_pattern.favorite_takeout`** —— 用户已经习惯的店
   - 例：用户答了"黄焖鸡 / 西贝 / 瑞幸" → 中午外卖推 "黄焖鸡米饭（去皮鸡腿 + 半份米饭 + 加青菜）" 而不是泛推"沙县小吃"
   - **不要**推用户没填过的店作为首选

3. **本文件区块 A/B 的清单作为 fallback**：
   - 用户没填 pantry → 默认推区块 A 的便利店搭配
   - 用户没填 favorite_takeout → 默认推区块 B 的 8 类外卖

4. **同时遵守 `restrictions` 和 `preferences`**：
   - "乳糖不耐" → 替换牛奶为豆浆 / 燕麦奶
   - "不吃香菜" → 提醒用户点黄焖鸡时备注"不要香菜"

5. **避免重复推荐**：同一周计划里同一道菜不要每天都推。3 天周期内换一次组合。

## 使用约束

- 便利店 / 外卖搭配的 kcal 与蛋白质数据为粗粒度估算（基于 2025 年中国大陆常见 SKU），不替代精确营养标签；引用时需加"约""估算"，并提示用户以包装标签为准。
- 用户 profile 标记乳糖不耐 / 不吃香菜 / 严格素食 / 海鲜过敏 / 麸质不耐等饮食偏好或过敏时，LLM 必须在引用本文件前 mentally filter，不得直接照搬含禁忌成分的组合。
- 训练前 3-6 个月内有膝伤 / 腰伤 / 怀孕史 / 心血管问题用户：跳过 C 区高强度（C.3 / C.6 / C.9），建议先与医师 / 物理治疗师面诊。
- 当用户在 menstrual 或 luteal 后段时，LLM 优先从 C 区低-中强度选项里挑（C.1 / C.2 / C.4 / C.5 / C.7 / C.8），不主动推 C 区高强度。
- 本文件不得用于建议日总能量低于 1200 kcal 的方案；与 `cycle-rules.md` §跨阶段共通原则冲突时，以 cycle-rules.md 为准。商品规格、连锁品牌 SKU 会随时间变化，建议每年 review 一次。
- **kcal / 蛋白质估算偏向上限**：本文件里给出的数值倾向连锁标准 SKU 的高位，实际包装可能略低。引用时建议以"约"或"估算"措辞，并主动提示用户"以包装标签为准"。
- **C 区时长 = 主体训练时长**：每套时长不含完整热身与放松（合计另需 5-8 分钟）。用户实际可用时间紧张时，主体可按比例缩短组数 / 次数。
- **最低组合保底**：若用户三餐都从本文件最低 kcal 组合里挑（如 ~320 + 350 + 320 ≈ 990 kcal），LLM 必须主动建议加一份高蛋白加餐（如水煮蛋 2 个 + 一杯无糖酸奶）补足到 ≥ 1200 kcal。
