# 周期阶段建议规则

> **适用范围**：本文件**仅当 `profile.cycle.status ∈ {regular, irregular}` 时使用**。绝经 / 围绝经 / 哺乳 / 备孕 / 孕期 / 医疗特殊用户请走 `non-cyclic-rules.md`，不要套用本文件的 4-phase 结构。

本文件是 fat-loss-agent 在生成每周计划或回答日常提问时的结构化建议库，LLM 在运行时按当前周期阶段查阅本文件，将其内容融入回答。**周期阶段判定算法不在此文件定义** —— 完整算法、边界语义与时区处理请参见 `state/profile-schema.md` §周期阶段计算（唯一规范源）。结构化引用（章节锚点、内部传值）使用小写英文 key：`menstrual` / `follicular` / `ovulatory` / `luteal`；面向用户的自然语言中使用中文阶段名（月经期 / 卵泡期 / 排卵期 / 黄体期）。语气保持温和、专业，面向中文成年女性用户，**绝不羞辱身材或体重**，不诊断疾病，不鼓励极端节食。

## 月经期 (menstrual)

经期出血伴随铁流失，雌激素与孕激素均处于低位，疲劳、腹痛、情绪低是普遍生理反应，不是个人意志问题。

### 饮食重点

- 重点补铁：建议安排猪肝 / 鸡肝、瘦牛肉 / 瘦羊肉、鸭血、黑芝麻等富铁食物，每周 2-3 次。
- 搭配维生素 C 帮助铁吸收：餐后吃 1 个猕猴桃、半个橙子或几片番茄。
- 暖性食物为主：推荐生姜红糖水、热汤面、煮苹果、红枣小米粥；尽量避免冰饮、刺身、生冷沙拉。
- 碳水可比平时增加 10-15%（约半碗米饭量），缓解疲劳与情绪低落，不必担心这点增量。
- 经期前两天若腹痛明显，可饮用温热的红枣桂圆茶替代咖啡，减少子宫平滑肌刺激。

### 训练强度

- 全部降级：建议 20-30 分钟散步、阴瑜伽（猫牛式、婴儿式、束角式）、轻柔拉伸为主。
- 暂停 HIIT、负重深蹲、平板支撑、跑步与高强度核心训练，避免加重盆腔充血。
- 前两天若疼痛严重或出血量大，允许完全休息一天，不算"破戒"。
- 拉伸前可热敷小腹 5-10 分钟，让身体先暖起来再活动。
- 走路配速可比平时慢 10-15%，以"能轻松说话"为强度上限。

### 情绪应对

- 接纳疲劳烦躁是激素水平的正常波动，**不必责怪自己没动力**。
- 具体行为：睡前 1-2 小时减少屏幕蓝光（关手机或开夜览模式），让褪黑素自然分泌；用热水袋暖小腹 15 分钟；允许吃一小块 70%+ 黑巧克力。
- 这一周不安排重大决策、签合同、剧烈争论或新计划启动。
- 给自己一杯热饮 + 一本轻松的书的独处时间，比硬撑社交更有恢复效果。

## 卵泡期 (follicular)

雌激素稳步上升，能量、专注力与情绪走高，胰岛素敏感性最佳 —— **这是减脂效率最高的窗口**，也最适合启动新计划。

### 饮食重点

- 优质蛋白增量：建议鸡胸肉、虾、鸡蛋、希腊酸奶、北豆腐轮换，支持力量训练后肌肉合成。
- 增加纤维维持饱腹感：西兰花、菠菜、藜麦、燕麦、毛豆是日常可选项。
- 复杂碳水匹配训练量：红薯、糙米、全麦面包、玉米可作主食，运动日适度多放半碗。
- 餐次安排稳定，三餐 + 1 次小加餐（如水煮蛋 + 蓝莓）即可，无需特别"卡热量"。
- 这阶段身体对蛋白与碳水的利用效率高，减脂与增肌可以同时推进。

### 训练强度

- 训练高峰期：力量训练 / 抗阻训练优先，推荐"全身训练 3 次/周"模板。
- HIIT 一周 2 次：例如开合跳 4 组 × 30 秒，或波比跳 3 组 × 10 个，间歇 30-60 秒（关节敏感者把波比跳替换为登山者 mountain climber 或快走爬楼梯）。
- 可加 1 次中长跑（3-5 km）或 30 分钟跳绳，提升心肺。
- 强度可以接近个人上限，组间休息 60-90 秒即可。
- 训练前后各 5-10 分钟动态/静态拉伸，避免次日酸痛影响后续训练。

### 情绪应对

- 动力高峰，**适合启动新习惯**：早起、戒糖、新训练计划放在这一周更容易坚持。
- 把本周难度最大的训练或饮食挑战安排在卵泡期中段（约 D8-D11）。
- 记录每日小胜利（一句话日记或打卡）积攒长期信心。
- 社交意愿高，可以约朋友一起锻炼或去公园快走，把运动社交化。

## 排卵期 (ovulatory)

雌激素峰值，睾酮短暂上升，力量与爆发力达到全周期最强，但水分潴留略增、韧带相对松弛。

### 饮食重点

- 继续高蛋白：建议每公斤体重 1.2-1.5 g 蛋白，分布在三餐 + 训练后加餐。
- 抗氧化食物适度增加：蓝莓、紫薯、深绿叶菜（菠菜、油麦菜）、原味坚果（核桃、巴旦木）每日轮换。
- 轻微减盐：避免水煮预制菜、罐头、酱料过量，减轻排卵期短暂水肿。
- 多喝水：每天 1.5-2 L，少量多次，帮助代谢与体温调节。
- 不必为体重秤上短期 0.5-1 kg 上浮焦虑，这多是水分而非脂肪。

### 训练强度

- 力量爆发力最强 —— 是个人 PR 日：深蹲、硬拉、卧推可挑战重量，每组 4-6 次的低次数高重量组适合本期（仅适用于已有 6+ 个月规律力量训练经验者；新手保持 8-12 次中重量区间，安全优先）。
- HIIT、跳跃训练（跳箱、跳绳双摇）效果最佳，可比卵泡期再加 1 组。
- 警惕运动损伤：雌激素峰值让韧带相对松弛，热身建议拉长到 5-10 分钟动态拉伸 + 1 组空杠/轻哑铃/无负重热身组。
- 训练后做 5 分钟泡沫轴放松股四头与髂胫束，降低次日紧绷感。
- 不建议第一次尝试新动作（如新引体上拉变式）放在 PR 日，新动作放卵泡期更稳。

### 情绪应对

- 自信心高、社交活跃、表达力强，**适合演讲、面试、重要会议**。
- 警惕"过度承诺"：把"周末跑半马"或"明天开始裸辞健身"这类激情决定先睡一觉再确认。
- 这阶段适合录视频、拍训练记录、做阶段复盘，状态会比较上镜。
- 若伴侣关系紧张，本期沟通效率高，可以选择此时坦诚谈话。

## 黄体期 (luteal)

孕激素主导，基础体温升高约 0.3-0.5 °C，食欲增加、易水肿、PMS（经前综合征）风险上升 —— 这是全周期最难的阶段，目标是"稳住"而非"突破"。

### 饮食重点

- 应对食欲增加：优先**高蛋白 + 高纤维**抗饿组合（鸡蛋、燕麦、毛豆、奇亚籽、希腊酸奶）。
- 补镁改善 PMS 与睡眠：深绿叶菜、南瓜籽、黑巧克力（70%+ 一小块）、香蕉每天选一两样。
- 减盐 + 减加工糖应对水肿与情绪波动；想吃甜时换成莓果、苹果、奇异果搭配一小块黑巧。
- **不要节食**：黄体期热量需求实际比平时高约 100-200 kcal，硬扛容易触发暴食反弹。
- 晚餐建议比平时提前 30-60 分钟，给基础体温升高的身体留出消化窗口，改善睡眠。

### 训练强度

- 中等强度为主：力量训练降低重量 10-20%，HIIT 时长或组数减半。
- 推荐稳态有氧：快走 30-40 分钟、瑜伽流、椭圆机 30 分钟、普拉提、室内骑行。
- 避免极限挑战：基础体温升高让心率与体感更累，硬上容易过度训练。
- 训练目标改为"维持习惯 + 出汗 30 分钟"，不追求新 PR。
- 若某天实在没动力，建议改成 20 分钟散步 + 5 分钟拉伸，保留运动节律即可。

### 情绪应对

- PMS 是激素事实，不是个人意志问题，**接纳而非羞辱**。
- 保证 7-8 小时睡眠（黄体期睡眠效率本就下降），需要的话提早 30 分钟上床。
- 减少咖啡（每日 ≤ 1 杯，14:00 后不喝），避免心悸加剧 PMS。
- 提前给自己留出独处时间，把高强度社交或加班尽量错开。
- 若有暴食冲动，先吃一份高蛋白（如水煮蛋 2 个 + 一碗燕麦）再决定下一步，多数冲动会缓解。

## 跨阶段共通原则

- 任何阶段都不建议单日 < 1200 kcal 饮食 —— 会压低基础代谢、增加暴食与反弹风险。
- 本 skill 不诊断疾病：PCOS（多囊卵巢综合征）、子宫内膜异位、甲状腺异常等不应由本 skill 给"判断"，出现疑似症状一律建议线下就医。
- 训练前 5-10 分钟动态拉伸 + 训练后 5-10 分钟静态拉伸是默认要求，所有阶段不省略。
- 出现以下情况建议主动找妇科医生评估：经期持续 > 7 天、突然停经 > 3 个月、周期 < 21 天或 > 40 天、非经期出血、痛经突然加重（如止痛药剂量需加倍 / 连续 ≥ 2 个周期请假休息 / 影响日常工作或睡眠）。
- 当用户表达"暴食 / 催吐 / 极度低卡 / 越瘦越好 / 不吃就开心"等信号时，**暂停**给出减脂建议，温柔询问近况，必要时建议寻求心理咨询或精神科就诊。
- 哺乳期 / 备孕 / 已知慢性病 / 正在服药（如复方口服避孕药、抗抑郁药、甲状腺药等）的用户，本 skill 提供的建议仅作生活方式参考，**必须先咨询主治医生**再执行。其中**复方口服避孕药使用者周期不再有真实卵泡期/黄体期切换**，本文件的阶段化建议不适用，建议改用通用减脂原则（高蛋白 + 高纤维 + 适度赤字 + 规律训练）。
- 当 profile.json 中周期数据已过期（按 `state/profile-schema.md` 的"陈旧数据检测"规则），本文件的阶段化建议暂不启用，仅提供保守的通用建议，并提醒用户更新 `cycle.last_period_start`。
- **健康优先于体重**：当用户因被拒绝某极端要求（如 < 1200 kcal、ED 信号、过训方案）而挫败时，本 skill 始终把"经期规律 / 情绪稳定 / 睡眠 / 长期可持续"放在体重数字之前。瘦得快但月经紊乱、情绪崩溃、暴食反弹的方案，不是好方案。
