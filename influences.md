# 参考博主与方法论

本文件汇总用户认可的几位中文女性健身 / 减脂博主的核心理念与方法。LLM 在生成饮食或训练建议时，可优先借鉴这些博主的**语气与方法论**，让回答更贴合用户已经认同的范式。

## 使用原则（必读，不可绕过）

1. **不直接代博主"说话"**：不要生成"小李不吃说……"或"欧阳春晓建议……"这种引语，避免误传。**可以说**"参考小李不吃式不节食理念"或"沿用欧阳春晓推崇的复合动作思路"。
2. **硬约束优先于博主**：任何博主方法若与 `cycle-rules.md` §跨阶段共通原则（1200 kcal 底线、ED 警示、不诊断、医疗免责）冲突，以本 skill 硬约束为准。
3. **个体差异**：博主方法是参考，不是处方。建议时必须结合 profile（年龄、周期、体脂、训练史 `training.history`、健身房状况 `training.gym_access`）。
4. **不复读全篇**：使用本文件时只引用 1-2 个最贴当下场景的点，不要把所有理念堆给用户。

## 小李不吃：不节食、关系优先

**核心理念**：
- 反对把每餐 kcal 算到小数点；粗略估算 + 长期看大方向
- 三餐规律 + 必要时 1-2 顿加餐，比"少吃多餐"或"OMAD"更适合女性
- 不戒碳水 —— 每餐都要有主食；极低碳水长期会影响月经
- 关注"与食物的关系" —— 把节食和暴食看成一对孪生失调，目标是回归"中立地吃"
- 不羞辱"破戒" —— 吃多一顿不是失败，下一顿正常吃就行

**落到本 skill 的应用**：
- 用户表达"又破戒了 / 没控制住"时，**先用此理念安抚**（配合 `cycle-rules.md` §luteal §情绪应对）
- 设计每周计划时，三餐分配优先于热量精确
- 用户问"早餐 / 午餐能不能不吃"时，倾向劝吃而非允许跳过

## 好人松松：简单可持续 / 生活化减脂

**核心理念**：
- 赤字 10-15% 足够；不追求 30%+ 的极端赤字
- 执行三件事：吃够蛋白质、减少加工糖与油、规律抗阻 + 适度有氧
- 正常吃饭 —— 外卖也能减脂，关键是会点
- 体脂率比体重更稳定 —— 体重周内波动 1-2 kg 多是水分，不焦虑
- 不一定要 IF / OMAD，三餐节奏对大多数女性更可持续

**落到本 skill 的应用**：
- 算每周 kcal 目标时按 BMR × 1.3 后再减 10-15%，落到约 1400-1800 kcal 区间
- 体重数据反馈时强调**趋势线**而非单次数值
- 用户被外卖困住时，按 `lifestyle-tips.md` §B 的"要 / 不要 / 加"框架给建议

## 欧阳春晓：女生必须练力量

**核心理念**：
- 女生只练有氧是误区 —— 加力量训练才能塑形，不是越瘦越好看
- 复合动作优先：深蹲、硬拉、卧推、引体、推举；每周 3-4 次力量为主
- 蛋白质要够（约 1.6-2.0 g/kg 体重）才能长出肌肉感
- 女生不会"练得像男生" —— 女性睾酮天然低，正常训练量不会粗壮
- 渐进负重 + 记录训练日志，看长期增长

**落到本 skill 的应用**：
- 卵泡期 / 排卵期推荐力量训练时，结合"复合动作 3-4 次/周"思路（`cycle-rules.md` §卵泡期 §训练强度）
- 用户表达"怕练力量变粗"时，引用此理念安抚（**不要替欧阳春晓直接说话**）
- 高蛋白目标 1.6-2.0 g/kg 是 luteal 食欲管理（`cycle-rules.md` §luteal §饮食重点）的重要基础
- 健身房状况 (`training.gym_access`)：
  - 有月卡：优先推杠铃 / 哑铃复合动作
  - 没办过：推家庭徒手 + 弹力带 + 单哑铃替代（参考 `lifestyle-tips.md` §C）

## 张俪：薄肌审美 + 自律低糖生活方式

> ⚠️ 我对张俪公开内容的理解概括如下，**未必每条都精确**，请你校准。

**核心理念**：
- 追求**薄肌身材**（slim + toned），**不追求大肌肉块感**——线条 + 流畅度优先
- 普拉提 + 瑜伽为主，配合慢跑 / 拉伸；不强推大重量力量
- 早睡（晚 11 点前），把睡眠当成保养的一部分
- **控糖 / 抗糖**：公开提过控制添加糖、奶茶 / 甜饮料基本戒
- 烹饪偏清淡：少油少盐少糖，蒸 / 煮 / 烤为主
- 自律但不极端：不鼓吹 OMAD / 极端节食，强调可持续 + 不反弹

**落到本 skill 的应用**：
- 用户表达"想要瘦但不要变粗"时，引用此理念安抚（**不要替张俪说话**）—— 解释普拉提 + 中等重量 + 高蛋白的组合不会让女性变粗
- 用户经常喝奶茶 / 含糖饮料抱怨减脂慢时，按此理念给出温和的减糖建议（不是戒到 0，是替换为无糖燕麦拿铁 / 茶 / 苏打水）
- 推训练侧重时，**普拉提 + 瑜伽 + 慢跑** 是张俪式审美对应的组合（参考 `weekly-split.md` 模板 B 上下半身 split + 有氧）
- 强调睡眠优先级：用户为了挤训练牺牲睡眠时，按此理念劝改

## 体态大师：体态修正 / 假胯宽 + 瘦腰大法 创始人

**背景**：
- 国家认证康复医学治疗师 + 体态学院资深评估师 + 体态矫正师
- 抖音号 `titaidashi`，323 万粉，1224 万赞
- 10 年间 1 对 1 评估超过 4 万人
- App 同名"体态大师"

**研发的概念 + 核心方法**（公开的）：
- **假胯宽**（2016 年首创概念）：胯宽位置在大腿根部突兀外突，缩短腿视觉长度。后天形成可矫正
- **瘦腰大法**：用户反馈训练后腰围缩小 2-8 cm；针对"练粗腰"问题
- **假性双下巴**：不是脂肪，是肩颈和体态问题导致
- **卧蚕训练**、**面部训练方案**：面部肌肉激活
- 理念："主流健身训练以健康和体能为主，不关注男女形体差异——女性需要专门的形体美学训练"

**6 个矫正假胯宽动作**（被国家体育总局官网收录）：
1. **臀桥** — 25 次/组 ×3 组
2. **跪姿后抬腿** — 25 次/腿 ×3 组
3. **站姿侧抬腿** — 15-20 次/侧 ×4 组
4. **宽距深蹲** — 脚尖与膝盖同向，膝不过脚尖；30 次 ×2 组
5. **消防栓式** — 跪姿大腿外抬，顶部停顿；15-20 次/侧 ×4 组
6. **鸽子臀部伸展** — 拉伸动作，30 秒/侧 ×3 组

**落到本 skill 的应用**（关键差异化）：
- **骨盆前倾用户**（profile.body_status.posture_concerns 命中）→ 重点推上面 6 个动作（特别 1/2/3/5 强化臀肌弱区）
- **假胯宽 / O 型腿 / X 型腿 用户** → 站姿侧抬腿 + 消防栓式作为臀中肌激活，蚌式可作替代
- **圆肩驼背 / 头前引用户** → 引用体态大师的"颈肩问题导致假性双下巴"理念，**强化背肌 + 拉伸胸前 + 颈深屈肌训练**
- **用户问"为什么练了反而粗腰 / 大腿"** → 引用她"主流健身忽略男女形体差异"理念，安抚 + 转向形体美学路径（小重量 + 体态优先）
- **特别：女性追求"线条而非围度"** → 体态大师的整套方法更贴合"薄肌审美 + 形体修正"路径，与张俪审美互补

**与其他参考博主的搭配位置**：
- 偏**美学 + 体态优先** → 体态大师 + 张俪
- 偏**饮食可持续** → 小李不吃 + 修贤 + 好人松松
- 偏**力量增肌** → 欧阳春晓

LLM 生成训练计划时遇到体态问题，**优先**借鉴体态大师的具体动作清单（参考 `exercises.md §禁忌动作映射 §体态问题`）。

## 修贤：放纵饮食法 / 明星营养师

> ⚠️ 以下来自媒体公开报道 + 她著作《放纵饮食法》的豆瓣评论及书评摘要。**不是直接引用她的话**，是方法论提炼，**请你校准**。

**背景**：
- 健身教练出身（大四考健身教练证），健美比赛参赛者
- 抖音 400 万+ 粉丝（不是小红书为主），疫情起步
- 服务过明星：江映蓉、杜海涛
- 累计指导 53 万人减脂（截至 2024 年 5 月）
- 著作《放纵饮食法》

**核心理念**（来自她的方法论）：
- **"放纵" ≠ 节食**：基于日常饮食习惯的细微调整控制热量，"吃"的反义词不是"饿"
- **进食顺序**：餐前 1 大杯水（小口慢喝）→ 蔬菜打底狂吃 → 高蛋白（掌心大的肉 / 蛋 / 海鲜）→ 拳头大的杂粮主食 → 饱了停筷
- **碳水分级**：避开"糖油混合物"（油条 / 手抓饼 / 油炸糖食）；土豆 / 红薯 / 糙米 / 燕麦放心吃
- **周期性放纵策略（核心）**：每周 1 顿放纵餐（炸鸡 / 奶茶不忌口），反而促代谢、打破平台期 + 心理可持续
- **放纵餐执行细节**：餐前猛灌水 + 先吃高蛋白，骗过身体狂消耗

**落到本 skill 的应用**：
- 用户挫败"破戒了"时：引用修贤"放纵不等于失败"+ 周期性放纵理念安抚（**不要直接引语**）。下一顿正常吃即可
- 设计周计划时可主动建议 "1 顿 cheat meal" 选项（如周六晚），与"5 餐安全 + 1 餐自由"框架对应
- 给具体食物建议时引用修贤式**进食顺序**：水 → 蔬菜 → 蛋白质 → 主食。这套对女性饱腹感和血糖稳定都好
- 用户问"能吃 X 吗"涉及糖油混合物（油条 / 烤冷面 / 手抓饼 / 蛋糕）时：按此理念劝"不戒，但日常少；要吃放周末放纵餐"
- 平台期久攻不下时：建议加 1 次放纵餐（前提 ED 信号未触发）刺激代谢
